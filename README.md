# 청약·분양 한눈에 (LH청약플러스 · 청약홈 통합 뷰어)

LH청약플러스(임대·분양)와 청약홈(민간·공공분양) 공고를 한 화면에서 정리해 보여주고,
청약·분양으로 아파트를 구입하는 모든 방법을 가이드로 제공하는 반응형 웹앱입니다.

## 빠른 실행 (설치 불필요)

Python 3.9+ 만 있으면 됩니다. (Node 등 추가 설치 불필요)

```
# 방법 1) 윈도우 탐색기에서 더블클릭
start.bat

# 방법 2) 터미널에서
python server.py
```

브라우저에서 **http://localhost:8000** 접속.

> 서비스키가 없으면 자동으로 `data/` 폴더의 **샘플 데이터**로 동작합니다. 화면 우측 상단 배지에서 현재 데이터 출처(샘플/실시간)를 확인할 수 있습니다.

## 실데이터(공공 OpenAPI) 연동

1. [공공데이터포털](https://www.data.go.kr) 회원가입 후 아래 서비스 활용신청 → 서비스키 발급
   - **한국부동산원 청약홈 분양정보 조회 서비스** (APT 분양정보)
   - **LH 분양임대 공고 조회 서비스**
2. `config.example.json` 을 `config.json` 으로 복사하고 키 입력:

```json
{
  "applyhome_service_key": "발급받은_청약홈_키",
  "lh_service_key": "발급받은_LH_키",
  "port": 8000
}
```

3. 서버 재시작. (또는 환경변수 `APPLYHOME_SERVICE_KEY`, `LH_SERVICE_KEY` 사용 가능)

> `config.json` 은 `.gitignore` 에 포함되어 키가 외부로 노출되지 않습니다.
> API 호출이 실패하면 자동으로 샘플 데이터로 대체되어 화면은 항상 동작합니다.

## 주요 기능

- **대시보드**: 전체/접수중/접수예정/마감임박 통계, 7일 내 마감 공고, 곧 시작 공고, 유형별 공고 수
- **공고 목록**: 출처(청약홈/LH)·유형 필터, 검색, 접수중만 보기, 마감임박/최신순 정렬, D-day 표시
- **구입 가이드**: 청약통장 → 순위·가점제 → 분양/임대 전 유형 → 특별공급 → 소득·자산 기준 → 신청 절차 체크리스트

## 폴더 구조

```
apt/
├─ server.py              # Python 표준 라이브러리 백엔드 (정적 서빙 + OpenAPI 프록시)
├─ start.bat              # 윈도우 실행 런처
├─ config.example.json    # 서비스키 설정 템플릿
├─ data/                  # 샘플 데이터(키 없을 때 사용)
│  ├─ sample_applyhome.json
│  └─ sample_lh.json
└─ web/                   # 반응형 프론트엔드 (빌드 불필요)
   ├─ index.html
   ├─ css/styles.css
   └─ js/{app.js, guide.js}
```
